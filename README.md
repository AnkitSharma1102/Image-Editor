# 🖼️ Image Editor (Canvas Project)

A simple Image Editor built using HTML, CSS, and JavaScript.  
This project is part of my learning journey where I focus on building projects instead of just watching tutorials.

It uses the Canvas API to apply real-time filters and allows users to download the edited image.

---

## 🚀 Live Demo  
👉 Live Link:https://ankitsharma1102.github.io/Image-Editor/

---

## ⚙️ Features

- Upload image from device  
- Real-time filters:
  - Brightness  
  - Contrast  
  - Saturation  
  - Hue Rotation  
  - Blur  
  - Grayscale  
  - Sepia  
  - Opacity  
  - Invert  
- Preset filters (Drama, Vintage, Old School, B&W, etc.)  
- Reset all filters  
- Download edited image  
- Built using Canvas API  

---

## 🧠 How It Works

- Image is drawn on `<canvas>`  
- Filters are applied using `canvasContext.filter`  
- Sliders update filter values in real-time  
- Final image is exported using `toDataURL()` and downloaded
